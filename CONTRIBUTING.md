# Contributing to the Prosthetic Access Atlas

We welcome contributions from researchers, clinicians, data scientists, advocates, and community members.

## Ways to Contribute

1. **Report Access Gaps** — Open an Issue with community name, state, distance to nearest provider
2. **Provide Provider Data** — Submit NPI number, service area, certification
3. **Improve Analysis** — Fork, enhance scripts, add data sources, create visualizations
4. **Advocate** — Share with policymakers and public health officials

## Data Standards

Provider CSV format: facility_name,category,address,city,state,zip,latitude,longitude,status,region,notes
Status values: verified | suspected | gap | outdated

## Code Style
- Python 3.10+, PEP 8

## Review Process
All contributions reviewed before merging. Provider data requires independent verification.
